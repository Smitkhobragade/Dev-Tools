# 🔧 Dev-Tools

Welcome to **Dev-Tools** — a curated collection of simple, useful, and fast tools for developers. Whether you're formatting JSON, generating UUIDs, or decoding JWTs, this is your all-in-one toolbox to speed up your workflow.

## 🚀 Why Dev-Tools?

- ✅ No login or ads
- 🧩 Fast, focused, and minimal
- 🌍 Open source & community-driven
- 💡 Built by devs, for devs

## 🧰 Available Tools (and more coming soon)

- JSON Formatter
- Regex Tester
- UUID Generator
- Base64 Encoder/Decoder
- JWT Decoder
- Lorem Ipsum Generator
- Color Picker
- Markdown Previewer
- Timestamp Converter
- And much more...
